On the Subject of Wire Sequence Sequence

Once an idea has taken hold of the brain it’s almost impossible to eradicate. An idea that is fully formed, fully understood—that sticks, right in there somewhere.

  • Within this module there are several component panels, but only one component panel is visible at a time. Switch to the next component panel by using the down button and the previous panel by using the up button.
  • Nested within each component panel are several wire panels with the same up and down capability.
  • Do not switch to the next wire panel until you are sure that you have cut all necessary wires on the current wire panel. Do not switch to the next component panel until you are sure that you have cut all necessary wires across all wire panels of the current component panel.
  • Cut the wires as directed by the following table. Wire occurrences are cumulative over all wire panels within each component panel, but the cutting rules vary by component panel.
