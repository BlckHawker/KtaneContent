On the Subject of Symbol Pad

And I thought that Keypad would be the only time that I’d have to relay symbols.

On the module are nine keys containing symbols.

Act as if each 4x4 grid is overlapping each other. Select the keys in reading order starting from the nth cell. The nth cell is the total number of ports present on the bomb. If the port count is above sixteen, subtract sixteen until it isn’t. If it is 0, start at cell 1.
