Apêndice SQUARE: Identificação das variações de
Quadrados Coloridos

If there are 4 of the 5 colors (Red, Yellow, Blue, Green, and Magenta) occurring exactly four times each with exactly 1 color missing:
Bicolored Squares
Se uma cor aparecer menos vezes que qualquer outra, and upon the first successful press, the pressed button turns white:
Quadrados Coloridos
Se três cores ocorrerem exatamente 2 vezes cada:
Quadrados Pigmentados
Se quatro cores ocorrerem exatamente 1 vez cada:
Quadrados Tingidos
If there are exactly 3 colors that occur only once:
Isocolored Squares
Se existirem 16 cores diferentes:
Quadrados Multicoloridos
Se uma cor aparecer menos vezes que qualquer outra, and upon the first successful press, the pressed button turns black:
Not Colored Squares
If there are 16 squares of a uniform color:
Overcolored Squares
