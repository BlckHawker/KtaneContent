On the Subject of Pinches

Trust me, this is just practice for Reaching the Top of something far greater.

  • When this module activates, climb the module by selecting a pinch on the bottom row and continuing to select (orthogonally or diagonally) adjacent pinches. Your climb must end at the top row.
  • To plan out your climb, take the name of the last module solved, or the name of this module if there aren’t any solved modules.
  • For each letter in the module’s name, increment the score of the category in which it can be found. When this is done, set every category’s score to itself modulo 5.
  • For each category score N, N pinches of any color within that same category MUST be used.
  • If the sum of all category scores is not at least 4, add 1 to the score of any category with a score of 0.
  • Scores persist between activations.
  • If the most recently solved module has the same name as this module’s previous activation, cycle the category scores one to the right instead.
  • Attempting to grab onto a non-adjacent pinch or selecting a pinch in a way that does not abide by the category scores sends you tumbling to the bottom, forcing you to restart the climb.
