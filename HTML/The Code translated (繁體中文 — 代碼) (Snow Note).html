關於【代碼】模組

是什麼種類的代碼？說不準是文字代碼!也可能是數字代碼!它甚至可以是字母數字代碼!! FFFFFFFFFFFFFFFFFFFFFFFFFF

要拆除模組，你需要輸入正確的代碼。取上方顯示的數字，對數字進行在下表中首個符合條件的操作，然後輸入計算結果後按【S】按鍵提交。【C】按鍵為清除先前的輸入。


決定代碼
如果序號的首個數字和最後一位數字相等，且沒有電池，提交顯示的數字。
否則，如果有CLR指示燈，把顯示的數字除以 8 。
否則，如果序號包含“X”、“Y” 或 “Z”，把顯示的數字除以 20 。
否則，如果有 5 個以上的埠口，把顯示的數字除以 30 。
否則，如果沒有電池，把顯示的數字除以 42 。
否則，如果亮起比熄滅指示燈多，把顯示的數字除以 69 。
否則，把顯示的數字除以 3 。
  • 如果結果出現小數，只需輸入整數部分。
