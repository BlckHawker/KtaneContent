Keep Talking and Nobody Explodes Mod 紅色暗号

モジュール詳細：紅色暗号

この混乱は俺の墓石に刻まれるだろう……。

モジュールは、3つのディスプレー、キーボード、2つの矢印、及び現在のページを表示するボタン(送信ボタン)で構成されている。

右の矢印を押すと、次のページに移動できる。左の矢印を押すと前のページに移動できる。ページは全部で3ページある。

ステップ1：転置半ポリュビオス暗号

このステップでは、暗号化された単語と、1ページ目の2つのキー（それぞれ分離キーと転置キーと呼ぶ）を使用する。

始めに、分離キーを取得し、最初に出現したものを除いた重複する英字を削除しつつ、半ポリュビオス暗号表の左半分に読み順で入力する。残りのアルファベットを一文字ずつ加え、足りなくなったら右半分に移動する。半ポリュビオス暗号表は、下図のような配置である。

1234512345
1
2
3####
LR

暗号化された単語の各英字を三つの座標に変換し、左右、段、列の順番に上から下に記録し、直前の英字の右隣に並べる。

その後、転置キーの各英字ごとに、それを半ポリュビオス暗号表から探し、先程作成した6×3座標行列に対して以下の処理を行う。

  • これが奇数回目の処理の場合、その英字がある段と同じ行列内の行を、列座標の数字だけ左にずらす。
  • この際、2段目にある3と3段目にある4と5は動かしてはいけない。動かせる数字のみに注目して処理すること。

ステップ2：二重三要素リフレクター暗号

このステップでは、2ページ目の3つのキーワードと、暗号化された最初の単語が必要になる。最初2つのキーワードは、2つの三要素リフレクターを作成するために使用され、3つ目のキーワードは、各暗号化ステップ完了後にリフレクターをシャッフルするために使用される。

三要素リフレクターを作るには、キーワードとアルファベットを順に連結して文字列を作り、重複する英字を先頭を除いてすべて取り除き、その文字列を中央に穴のあいた3×9の格子に読み順で並べる。一番目のキーワードから作られた三要素リフレクターは上段のリフレクターであり、二番目のキーワードから作られたものは下段のリフレクターである。

暗号化された単語の各英字に対して、上段のリフレクターからその英字を見つけ、次に下段のリフレクターから同じ位置にある英字を見つける。この英字は一つ目の中間文字である。上段のリフレクターからその文字を見つけ、次に下段のリフレクターから同じ位置にある文字を見つける。この英字は二つ目の中間英字である。その後、下段のリフレクターと同じ位置にある英字を探し、復号された単語の英字を見つける。

各リフレクター内で、中間文字を見つけた列と段を記録する。一つ目の中間文字は下段のリフレクターから、二つ目の中間文字は上段のリフレクターで見つかったことを念頭に置くこと。次の英字を復号する前に、両方のリフレクターを次のように再配置する。

  • 現在の位置にある3つ目のキーワードの英字に対し、アルファベット上の位置の数字を三進数で取得する。
  • これが奇数回目の処理の場合、取得した値の上位2桁を三進数で取得し、一つ目の中間文字が見つかった段をその数だけ左にずらし、二つ目の中間文字が見つかった列を最下位桁と同じ数だけ下にずらす。
  • これが偶数回目の処理の場合、一つ目の中間文字が見つかった列を、取得した値の最上位桁と同じ数だけ下にずらし、下位2桁を三進数で取得し、二つ目の中間文字が見つかった段をその数だけ左にずらす。
  • リフレクターを動かす際に、穴は移動させないこと。

ステップ3：カエサルシャッフル暗号

このステップでは、ステップ2の暗号化された単語と、3ページ目の2つの5文字のキーワードが必要になる。

両方のキーワードの各英字について、以下のステップを実行する。

  • 暗号化された単語のX番目の文字の後ろに目印を置く。Xは、現在の位置にある一つ目のキーのアルファベット上の位置の数字を5で割った余りに1を足した値に等しい。目印の左右にある2つの文字列を入れ替える。
  • 左側にある部分文字列を、現在の位置にある二つ目のキーのアルファベット上の位置の値だけ前方にカエサルシフト(A -> B)する。目印を削除する。

例

暗号化された単語: GBVNWH
キーワード1: GROSS
キーワード2: CRASS

GBV.NWH -> NWH.GBV -> QZK.GBV -> QZKGBV

QZKG.BV -> BV.QZKG -> TN.QZKG -> TNQZKG

T.NQZKG -> NQZKG.T -> ORALH.T -> ORALHT

ORALH.T -> T.ORALH -> M.ORALH -> MORALH

MORAL.H -> H.MORAL -> A.MORAL -> AMORAL

復号された単語: AMORAL

復号された単語を取得したら、それを送信する。入力を開始すると、全てのディスプレーが暗転し下のディスプレーに入力された文字が表示される。

入力を消去するには、いずれかの矢印をクリックする。ページが移動し、入力した文字列が消去される。6文字を超える入力はできないようになっている。

入力に問題がなければ、「SUB」と書かれたボタンを押して、回答を送信する。ミスが記録された場合、モジュールの最初のページに戻るが、暗号の再生成はされない。
