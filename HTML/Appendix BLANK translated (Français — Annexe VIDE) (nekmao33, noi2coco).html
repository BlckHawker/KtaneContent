Annexe VIDE : Identification des modules vides

Quelqu’un à Bomb Corp. a été un peu paresseux sur les graphismes.

Vous avez peut-être remarqué que certains modules semblent (du moins au début) vides. À première vue, tous les modules de cette variété sont indiscernables*. Suivre les instructions ci-dessous afin de déterminer quel module vous avez devant vous.

* Note : Le module Charnières a une ou plusieurs charnières manquantes dans le cadre du module, mais est autrement vide.

  1. Sélectionner le module (mais ne pas appuyer dessus).
    • Si le module commence à se transformer, vous êtes en présence du 9-cube.
    • Si le module est maintenant entouré de rouge, c’est qu’il s’agit de ...?.
  2. Faire des va-et-vient avec le curseur sur le module.
    • Si le module comporte des motifs de points, il s’agit du Braille.
    • Si le module comporte des formes, il s’agit des Polygones.
    • Si le module comporte des chiffres et une forme, il s’agit de Caché à la vue de tous.
    • Si le module montre la lettre D, il s’agit du D.
    • Si vous entendez une tonalité, il s’agit de l’Allée des Sourds.
    • Si vous entendez un cliquetis qui se produit lorsque vous passez le curseur sur un bouton mais que vous n’avez aucun retour visuel, il s’agit de l’Impasse. Suivez simplement les instructions de la première page de ce manuel.
  3. Appuyer et maintenir le module.
    • Si le module s’est brisé comme du verre, il s’agit d’Une Erreur. Il est possible de le relâcher à tout moment.
    • Si le module commence à faire des bruits de tic-tac, il s’agit du Code Frappé. Il est possible de le relâcher à tout moment.
    • Si deux zéros commencent à clignoter sur le module, il s’agit de 42. NE PAS RELÂCHER jusqu’à ce que le manuel de ce module vous indique de le faire.
    • Si trois zéros commencent à clignoter sur le module, il s’agit de 501. NE PAS RELÂCHER jusqu’à ce que le manuel de ce module vous indique de le faire.
