モジュール詳細：求人応募

母親に就職しなさいと言われて腹を立てているのかい？就職活動を始めようぜ - バーチャルでな…

あなたは決心した。今日がその日だ…。「求人に応募して、KTANEは引退するぞ。」なんと素晴らしい！始めよう。

これは実際の時間とエッジワークに基づいて、求職者が求人応募書に記入する正しい名前、雇用形態、仕事を選択するモジュールである。

まず、爆弾が開始された実際の曜日を確認する。次に、指定した曜日に対して設定されたケースに従う。

一週間

月曜日

ガーフィールドは月曜が嫌いだ…

  1. 求職者「Emily」を選択する。
  2. 爆弾にシリアルポートがある場合、雇用形態「Flexible」を選択する。
  3. それ以外の場合、雇用形態「Part-Time」を選択する。
  4. 仕事「Usher」を選択する。

火曜日

  1. 求職者「Jacob」を選択する。
  2. 爆弾にバッテリーがちょうど2つある場合、雇用形態「Full-Time」を選択する。
  3. それ以外の場合、雇用形態「Flexible」を選択する。
  4. 仕事「Lifeguard」を選択する。

水曜日

  1. 求職者「William」を選択する。
  2. 爆弾のシリアルナンバーに偶数が含まれている場合、雇用形態「Part-Time」を選択する。
  3. それ以外の場合、雇用形態「Full-Time」を選択する。
  4. 仕事「Retail」を選択する。

木曜日

  1. 求職者「Olivia」を選択する。
  2. 爆弾にステレオRCAポートがある場合、雇用形態「Full-time」を選択する。

  5. それ以外の場合、仕事「Barista」を選択する。

金曜日

  1. 爆弾のシリアルナンバーに「AEIOU」のいずれかが含まれている場合、求職者「Danny」を選択する。
  2. それ以外の場合、求職者「Ava」を選択する。
  3. 爆弾に点灯していないCARインジケーターがある場合、雇用形態「Flexible」を選択する。
  4. それ以外の場合、雇用形態「Part-time」を選択する。
  5. 仕事「Musician」を選択する。

土曜日

  1. 求職者「Matthew」を選択する。
  2. 爆弾にPS/2ポートがある場合、雇用形態「Full-time」を選択する。
  3. それ以外の場合、雇用形態「Part-time」を選択する。
  4. 仕事「Babysitter」を選択する。

日曜日

そして第七日目に、神はその御行を終え、第七日目に休まれた。

  1. 求職者「Matthew」を選択する。
  2. 雇用形態「Flexible」を選択する。
  3. 仕事「Teacher」を選択する。

送信

設定中の全属性がそれぞれの曜日から割り当てられているものになったら、送信ボタンを押す。それでモジュールが解除される。
