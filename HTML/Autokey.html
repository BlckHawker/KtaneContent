On the Subject of Autokey

Calc it for 5 solves. Scratch that, 6.

Once the bomb’s timer matches the time on this module, it will solve. The countdown shows how long you have before this will happen (ignoring time increases from strikes).

When the module is 10 seconds from solving, it will play a warning sound, unless the mod setting to disable it is enabled.

You may also change the range of times that the module can choose from in the mod’s settings, notated as a percentage of the bomb’s initial time, from 0 to 100 percent. By default, the module will generate times between 25 and 75% of the bomb’s initial time.

Missions can enforce these settings, using the following syntax:

  • Replace X with a number to allocate different settings for each instance of Autokey (eg. [Autokey 3] will affect the third autokey on the bomb).
  • [Autokey X] earliest N: Set the earliest time the module can generate to N% of the bomb’s initial time.
  • [Autokey X] latest N: Set the latest time the module can generate to N% of the bomb’s initial time.
  • [Autokey X] soundoff: Disable sound.
