부록 SQUARE: 색 사각형 부류의 모듈 식별하기

만약 5개의 색깔 (빨강, 노랑, 파랑, 초록, 자주) 중 4가지가 정확히 4번 나타나고 1가지의 색깔만 없다면:
이색 사각형들 (Bicolored Squares)
만약 한 색깔이 다른 어떤 색깔보다도 적게 나타나며 (동률 없음), 성공적으로 눌렀을 때 버튼이 흰색으로 변한다면:
색 사각형들 (Colored Squares)
만약 3가지의 색깔이 정확히 2번씩 나타나고 2가지의 색깔이 정확히 5번씩 나타난다면:
탈색 사각형들 (Decolored Squares)
만약 4가지의 색깔이 정확히 1번씩 나타난다면:
변색 사각형들 (Discolored Squares)
만약 정확히 3가지의 색깔이 1번씩 나타난다면:
등색 사각형들 (Isocolored Squares)
만약 빨강, 파랑, 초록, 노랑, 자주 이외의 색깔이 있고, 중복되는 색이 없다면:
각색 사각형들 (Juxtacolored Squares)
만약 한 색깔이 다른 어떤 색깔보다도 적게 나타나며 (동률 없음), 성공적으로 눌렀을 때 버튼이 검은색으로 변한다면:
색 사각형들 아님 (Not Colored Squares)
만약 동일한 색의 사각형이 16개 있다면:
과색 사각형들 (Overcolored Squares)

만약 폭탄을 특정한 각에서 볼 때 사각형들의 색이 바뀐다면:
관점색 사각형들 (Perspecticolored Squares)
만약 빨강, 파랑, 초록, 노랑, 자주 이외의 색깔이 있고, 중복되는 색이 있다면:
묘비 미로 (Tombstone Maze)
만약 2가지의 색깔이 다른 어떤 색깔보다도 적게 나타난다면 (둘 간의 동률):
무색 사각형들 (Uncolored Squares)
만약 4가지의 색깔이 정확히 3번씩 나타난다면:
잡색 사각형들 (Varicolored Squares)
