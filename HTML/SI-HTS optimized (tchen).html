On the Subject of SI-HTS

The better you are at physics, the worse you’ll be at SI-HTS.

Step 1: Calculating the target value

Interpret each character in the serial number as base-36, and take that number modulo 5. Then, use the first row in the table below to translate the number into a score and calculate the sum. If the sum is between -1 and 1, inclusive, the target value is the sum. Otherwise, use the subsequent row and calculate the sum again. Repeat this process until you obtain a sum that falls between -1 and 1, inclusive. Use this as the target value.

+1 0 -1 -1 +1
+1 +1 0 -1 -1
-1 +1 +1 0 -1
The target value is 0.

Step 3: Calculating the input for the module

  • Calculate the sum of the scores for “Underhand” and “Flick”, and input the row with the lower sum. Input “Flick” in case of a tie.
  • Using the previously selected row, select the button in the column with the score that matches the target value from step 1. However, if the silhouette is upside down, use the up arrow instead of the down arrow, and vice versa.
