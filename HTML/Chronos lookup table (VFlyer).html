Original Manual

Using the name of the god displayed, obtain the equivalent name and trait to set on the module. The trait is determined by the material of the stopwatch used, with gray representing silver, and yellow representing gold.

The bottom-most arrows will adjust the Roman equivalent of the gods, the top-most adjusts their traits. To submit, select the stopwatch before pressing the submit button.

Submitting an incorrect combination will result in a strike if the timer is running. In addition, the module will strike after 2-5 seconds after activating the stopwatch if the module is not solved. Stopwatch conditions for when the module will strike are NOT shown in this manual.
