Who would think that wires this complex could totally refrain from the Venn Diagram?

White Wires

Red Wires

Blue Wires

  • Only blue or blue + white ->

  • Star + LED ->

Red + Blue Wires

  • Only blue + red ->

Letter meanings

  • C -> Cut
  • D -> Don’t cut
  • S -> If last serial # digit is EVEN, cut
  • P -> If there’s a parallel port, cut
  • B -> If there’s two or more batteries, cut
