モジュール詳細：ニュートリノ

亜原子パイのような光が目に入ったら、それはニュートリノだ...。

このモジュールは、目的地の惑星を表示するスクリーンと、6つの円形のボタンで構成されている。

成功のためのヒント

  • このモジュールを解除するには，太陽から太陽系の惑星に飛来した3つのニュートリノのフレーバーを特定する必要がある。
  • モジュール上部の3つのボタンは、3つのニュートリノと飛来開始時点における種類を表示している。
  • モジュール下部の3つのボタンは、押すと、ニュートリノの6つの組み合わせを順番に表示する。
  • モジュールを解除できる準備が整うまで、下部の3つのボタンや惑星名ディスプレーを押してはならない。
  • モジュールの回答を確定できたら、下部3つのボタンを押して順番に正しいニュートリノに設定し、惑星名のディスプレーを押して送信する。
  • もし、下の3つのボタンのいずれかを順番通りに押さなかったり、解答が間違っているときにディスプレーを押したりした場合、ミスが記録されるが、それ以外にモジュールに変更が加えられることはない。
  • 上部3つのボタンは操作できないため、初期ニュートリノは変更することができない。
  • ニュートリノの正しいフレーバーを決定するには，ニュートリノが目的地の惑星に到着するのにかかった時間を知る必要がある。マニュアルの次のセクションでは，3つのニュートリノそれぞれの移動時間を計算する方法を説明する。
  • ニュートリノにはe, μ, τからなる3つのフレーバーが存在する。また、各フレーバーごとに対応する反ニュートリノも存在し、これらは上部に線を引くことで表現する。
  • 同じフレーバーのニュートリノと反ニュートリノは、他の粒子および反粒子と同様に、互いに打ち消しあうことができる。
  • ニュートリノ3種の質量はほぼ区別がつかない(1.2 mu)ため、一定確率でフレーバーが変化することがある。この現象を「ニュートリノ振動」という。
  • ニュートリノは非常に高速で移動するため、移動する距離は実際の距離より縮んでしまう。これをローレンツ収縮という。そのため、同じ経路で同じ目的地に進んでいても、それぞれのニュートリノが移動する距離は異なることに気づくだろう。
  • このモジュールの解除において、相対性理論の計算を行う箇所があるが、値の単位を認識できないものがある。これはモジュールの解除に影響しない。

所要飛来時間の計算

  1. それぞれのニュートリノについて以下のステップを実行する。まず、運動量を決定しなければいけない。cはボタンの位置番号（左から1, 2, 3）に等しい。

    2. c a b 共通除数
    左から数えたボタンの位置 1 バッテリーホルダーの個数を10で割った余り ポートの個数を10で割った余り 301
    2 ポートプレートの個数を10で割った余り バッテリーの個数を10で割った余り 302
    3 点灯していないインジケーターの個数を10で割った余り 点灯したインジケーターの個数を10で割った余り 333

  2. 値a, b, cを結合し(abc)、この新しい3桁の数字を共通除数で割った余りを求める。その後、それを1000で割る。運動量は以下の式で求められる。%演算子は、左の数字を右の数字で割った余りを求めることを示す。
    p = 1.2 - ((abc % 共通除数)/1000)

  3. 次に、運動量をニュートリノの質量で割り、速さ(β)を求める。速さが1の場合は物理法則が壊れてしまったため、こっそりバレないようにそのまま送信ボタンを押すこと。以降の各計算は、四捨五入して小数第三位まで求めるようにする (例：1.87746 ≅ 1.877, 1.87756 ≅ 1.878)。
    β = p/1.2

  4. ローレンツ因子(ɣ)を計算する(sqrtは正の平方根を返す関数である)。
    ɣ = 1/sqrt(1-β*β)

  5. ニュートリノが進んだ収縮後の距離(L)を計算する(付録Aで太陽からの各惑星の距離を求めることができる)。
    L = 太陽からの距離/ɣ

  6. 所要飛来時間は、距離(L)を速さ(β)で割って求める。この計算のみ、小数点以下を切り捨てて最も近い整数にする。
    所要飛来時間 = L/β

  7. 所要飛来時間を10で割った余りに1を足す。混乱しないよう、この値を最終時間と呼ぶ。この値を次のセクションで使用し、ニュートリノの最終フレーバーを求める。

最終フレーバーの特定

  1. 以下の表で、ニュートリノの初期フレーバー書かれているセル群の中央のセルから開始する。
  2. このセルから、条件が正しければ、最終時間から1を引く。最終時間がゼロではない場合、一段下に進み上記のステップを繰り返す。表は必要に応じてループさせる。
  3. このステップで使用しているニュートリノが反ニュートリノの場合、表を一段下ではなく一段上に進む。
  4. 最終時間から引いた結果が0になるセルに到達したら、そのセルに対応するフレーバーが新しいニュートリノのフレーバーになる。反ニュートリノは、反ニュートリノのままフレーバーのみが変更される。
  5. 同じセルに2回到達すると、「一周した」とみなされる。周回数は条件の判定を行う前に増加する。

  3. 重要:最終ニュートリノのうち、二つが同じフレーバーのニュートリノと反ニュートリノのペア(例：eと線付きe)の場合、2つは互いに打ち消し合う。これらのニュートリノは空欄のままにすること。3つすべてのニュートリノが打ち消し合う場合、到達が最も早いニュートリノと反ニュートリノのペアのみで打ち消し合い、3つ目のニュートリノには影響しない。複数の打ち消し合いが発生する状況で同じフレーバーが同時に到着した場合、より右にあるニュートリノが打ち消される。
