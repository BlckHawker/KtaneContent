Keep Talking and Nobody Explodes Mod Quaver

モジュール詳細：Quaver

恒久的なNLNに設定したよ！待って、ランクが付かないってどういうこと？

  • このモジュールには、ディスプレーと矢印キーがある。
  • このモジュールを解除するには、上部のバーを埋める必要がある。

難易度設定

  • モジュールは難易度選択画面から開始する。
  • 処理担当者は3つの項目を自由にカスタマイズすることが出来る:
    1. Speed:矢印の落下速度(1.0x-2.0x)
    2. Difficulty:矢印の数(Normal, Hard, Insane, Expert)
    3. Per Column:列ごとに数えるか総数を数えるか(On/Off)
  • 左右の矢印キーは、現在選択されている項目を調整する。
  • 上下の矢印キーは、選択する項目を変更する。
  • スペースバーを押してゲームプレイに入る。

遊び方

  • 最初に上下の矢印キーでスクロール速度を調整する。
  • 落ちてきた矢印の数の総数を数える。
    • 「Per Column」をOnにした場合は列ごとに数える必要がある。
  • シークエンスが終了すると、モジュールは送信モードに入る。
  • 矢印キー/スペースバーを矢印の数分押す。
  • キーを押して2秒間放置すると、入力された数字が送信される。
    • キーを入力することで2秒のタイマーはリセットされる。
    • 数字は循環する。
  • 正しく数字が入力された場合は上部のバーにポイントが追加され、間違った数字が入力された場合はミスが記録される。
  • その後選択画面に戻る。

解除方法

  • モジュールを解除するにはS×D×Pの式を使用する。解除には100必要になる。

S = Speed [1.0x = 1, 1.5x = 1.5, 2.0x = 2]
D = Difficulty [Normal = 10, Hard = 20, Insane = 40, Expert = ∞]
P = Per Column [ON = 1, OFF = 0.5]
