О расшифровке цветов

Цвета, Саймон, что они значат?!

Модуль решается в 3 этапа. На каждом этапе присутствует 4х4 сетка-индикатор и основная сетка 6х6. Сетки состоят из пяти цветов: красный (К), зелёный (З), синий (С), жёлтый (Ж) и фиолетовый (Ф). На каждом этапе вам необходимо выбрать несколько столбцов и/или рядов на основной сетке. Чтобы выбрать стобец/ряд, нажмите на кнопку сверху/справа от нужного стобца/ряда соответственно. Чтобы определить правильные столбцы/ряды, следуйте следующим правилам:

  • Используя узор сетки-индикатора и информацию о бомбе, определите ряд в таблице для данного этапа.
  • Для определения нужных в данном этапе эллипсов диаграммы Венна, используйте цвета на сетке-индикаторе. Для каждого цвета, присутствующего одновременно на сетке-индикаторе и в определённом ранее ряду, возьмите эллипсы из колонок, соответствующих этим цветам в ряду. Используя полученные эллипсы, определите по диаграмме Венна номер списка условий для данного этапа.
  • В выбранном списке по порядку для каждого условия найдите и выберите в основной сетке ряд/столбец в котором оно выполняется, НО игнорируйте условия в пропускаемых секциях. Пропускаемые секции находятся по таблице, используя колонку номера этапа и ранее определённый ряд.
  • После решения трёх этапов модуль будет обезврежен.
