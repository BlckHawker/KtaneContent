Appendix 8TN5: Identifying Directing Buttons siblings

If each button is either black or white:
Uncoloured Buttons
If the buttons are flickering in unison:
Faulty Buttons
If there is a large button in the middle:
Surrounding Buttons
If each button is either orange, lime, cyan or pink:
Jumbled Buttons
If the grid changes with a wipe effect:
Switching Buttons
Otherwise:
Directing Buttons
