Keep Talking and Nobody Explodes Mod 42

À propos de 42

La réponse à la vie, l’univers, et le reste... 42.

Voir l’annexe VIDE pour l’identification des modules vides.

Ce module apparaît initialement vide. Dès la première pression sur le module, une tonalité est émise et, une seconde plus tard, le module commence à générer 10 numéros à deux chiffres. (Pour des raisons pratiques, la première étape génère toujours dix 00.) Ne PAS relâcher le module immédiatement, car cela enregistrerait une erreur.

Tous les autres cycles commenceront immédiatement à afficher les chiffres.

Si un autre module non-ignoré est résolu, les chiffres du cycle vont changer.

Pour désamorcer ce module, relâcher lorsque l’écran affiche 42. Relâcher sur un autre chiffre ou sauter une étape avec un 42 enregistrera une erreur. (Si tous les modules non-ignorés sont désamorcés et que ce module n’est pas encore résolu, tous les numéros deviennent valides. Relâcher à tout moment pour désamorcer).

Si le module est maintenu, il est possible de le relâcher pendant la pause pour éviter une erreur.
