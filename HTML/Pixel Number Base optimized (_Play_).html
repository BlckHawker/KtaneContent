On the Subject of Pixel Number Base

Because 216.

Convert each 4×4 grid’s row (both, in reading order) from binary to hexadecimal, where a black pixel represents a ‘1’, and a white pixel - a ‘0’. Submit the resulting 16-digit hexadecimal number.

Grid 1 : 1111 1001 0101 1000 → F958

Grid 2 : 0111 0000 0110 1000 → 7068

Grid 3 : 1011 0100 1001 0011 → B493

Grid 4 : 0001 0001 0111 1111 → 117F
