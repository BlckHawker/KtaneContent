Instruction Guide:

  1. press the top button.
  2. press the bottom button.
  3. press the top button if the last serial digit is odd; otherwise, the bottom button.
  4. press the bottom button if there are more ports than batteries; otherwise, the top button.
  5. press either button.
  6. press the button that was pressed first in the previous stage; if this is the first stage, press the top button.
  7. press the button you've pressed the least this stage so far; if it's a tie, press either button.
  8. press the bottom button if this instruction is in the first half of this row's instruction list; otherwise, the top button.
  9. press the top button if the seconds digit of the timer is currently even; otherwise, the bottom button.
  10. press the bottom button and ignore all further instructions from this row; instead, move to the row above and continue from the first instruction.
  11. press either button; if this is the first or second stage, ignore the last instruction of the next stage and instead press whichever button you didn't press here.
  12. press the top button if it has any matching letters with this stage's display word; otherwise, the bottom button.
  13. press the bottom button if it has any matching letters with the serial number; otherwise, the top button.
