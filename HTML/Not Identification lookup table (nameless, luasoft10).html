Keep Talking and Nobody Explodes Mod Not Identification

On the Subject of Not

Identify these lookup tables.

Possible forms:

  • Animatronic Identification (Ultimate Custom Night)
  • FNF Identification (Friday Night Funkin')
  • Item Identification (Minecraft)
  • Magisword Identification
  • Piece Identification (Composers)
  • Stand Identification (JoJo's Bizarre Adventure)
  • Touhou Identification
  • VTuber Identification

To solve the module, complete the four stages as follows.

Note:

  • Avoid pressing any keys not mentioned for any given stage as this may cause the module to strike.
  • The module keyboard support assumes you are using a QWERTY layout, some keys may not type their corresponding keys on the module if you use a different keyboard layout so be careful when inputting.

Stage 2:

Press the keys "01234" and take note of which digits they type, you will need to type the three-digit answer you get using these five keys.

Press the keys "5789". These keys type out the digits ABCD. The answer you need to submit for this stage can be calculated as follows:

(756A + 540B + 945C + 280D) % 1260, where % is the modulo operator (ie. remainder after division by 1260).

Stage 3:
