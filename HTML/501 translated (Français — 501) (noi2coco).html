Keep Talking and Nobody Explodes Mod 501

À propos de 501

Qu’est-ce que 501 a de si spécial, vous demandez-vous ? Eh bien, je ne le sais pas non plus !

Voir l’Annexe VIDE dans Impasse pour l’identification des modules vides.

Ce module apparaît initialement vide. Dès la première pression sur le module, une tonalité est émise et, une seconde plus tard, le module commence à générer 10 numéros à trois chiffres. (Pour des raisons pratiques, la première étape génère toujours dix 000.) Ne PAS relâcher le module immédiatement, car cela enregistrerait une erreur.

Tous les autres cycles commenceront immédiatement à afficher les chiffres.

Si un autre module non-ignoré est résolu, les chiffres du cycle vont changer.

Pour désamorcer ce module, relâcher lorsque l’écran affiche 501. Relâcher sur un autre chiffre ou sauter une étape avec un 501 enregistrera une erreur. (Si tous les modules non-ignorés sont résolus et que ce module n’est pas encore désamorcé, tous les numéros deviennent valides. Relâcher à tout moment pour désamorcer).

Si le module est maintenu, il est possible de le relâcher pendant la pause pour éviter une erreur.
