On the Subject of Polyhedron Preparation

painted rocks 🤤 😋

Welcome to the polyhedron kitchen! To solve the module, turn the given polyhedron into food.

  • The module will initially show the dish you are making at the bottom. After filling in the structure, the text will change to an ingredient.
    • If the placement is incorrect, the face will not be filled in with that ingredient and the module will strike.
    • The stage will automatically progress once all correct faces have been filled. The module will solve after 3 ingredients.
    • If the polyhedron is completely filled with ingredients before the module is solved, some faces will be randomly cleared.
  • You can rotate the polyhedron using the arrows in the corners.

Before adding ingredients, you must first make the structure of the dish. Every dish can be sorted into 6 main groups, each with a unique structure described in the first table on the next page.

  • The main face is the initial front-most face on the module and is denoted with a ring.
  • The opposite faces refers to the faces with the most number of faces away from the main face.
  • Adjacencies only include edge-to-edge connections.

After making the structure, you will start adding ingredients. Every food has a unique cooking process stated in the second table.

  • “Liquid” and “Solid” refer to the state of that ingredient.
  • Between ingredients, there might be instructions in ALL CAPS that can modify the state of all faces that aren’t the first ingredient.
    • COOK makes ingredients hot. CHILL makes ingredients cold.
    • If a face would be hot and cold at the same time, it is neither.

The correct faces to fill with the 3 ingredients will depend on the state of other ingredients and their states. All ingredients’ rules are described in the last pages.

Structures:

Plate: Pizza, Spaghetti, Toast
Add the main face, then add in the adjacent faces if doing so wouldn’t color in more than 25% of faces.
Stack: Sandwich, Kebab, Lasanga
Add the main and opposite faces.
Roll: Burrito, Churro, Sushi
Add all faces the same distance away from the main and opposite faces. If no faces apply, add all faces where the distance between the main and opposite faces are one off.
Bowl: Ice Cream, Honeycomb, Quiche
Add every face closer to the main face than the opposite face.

Processes:

Pizza Liquid, Solid, COOK, Solid
Spaghetti Solid, Liquid, CHILL, Solid
Toast Liquid, CHILL, Solid, Liquid
Sandwich Solid, COOK, Solid, CHILL, Liquid
Kebab Solid, COOK, Solid, COOK, Solid
Lasanga Liquid, Solid, COOK, Liquid
Burrito Solid, COOK, Liquid, Liquid
Churro Liquid, COOK, Solid, Solid
Sushi Solid, Solid, CHILL, Liquid
Ice Cream Solid, CHILL, Liquid, CHILL, Solid
Honeycomb Liquid, COOK, Liquid, COOK, Liqui
Quiche Liquid, COOK Solid, Solid

Cheese

Solid:

Egg

Solid:

Honey

Solid:
Place it in every face with more than 5 sides.		 Liquid:
Place it in every triangle face.

Icing

Solid:
Place it in every face an odd number of faces away from the main face.		 Liquid:
Place it in every face an even number of faces away from the main face.

Meat

Solid:
If any of the faces are sauce, fill in all faces adjacent to sauce. Otherwise, fill in all faces adjacent to the structure but not to any other food.		 Liquid:
Fill in all faces adjacent to faces you filled last stage.

Pepper

Solid:
If any faces are hot, place it in any triangles adjacent to hot faces. If not, place it in any triangle adjacent to the structure.		 Liquid:
Place it in every triangle face not adjacent to any other filled-in face.

Sauce

Solid:
Place it in every face adjacent to a hot face. If there are no hot faces, place it in every tile not adjacent to a cold face or the structure.		 Liquid:
Place it in every tile adjacent to a hot face, and every face adjacent to the structure.

Trash

Place it in every available face. State does not matter.
