Do not press the red square whenever you want! It will strike you!

Instead, wait until the number of solved modules is equal to the number of minutes remaining, then press the red square.

If the number of solved modules and number of minutes become equal, and another module is solved without the red square being pressed, it will still strike you! And scare you!

If this does happen, instead, double the time remaining for each strike that Red Square receives.

Some modules may be ignored. If an ignored module is solved when the red square was expecting input, it will not strike you, but input will still be expected.
