Components - Batteries

  • Set to ((batteries + battery holders + 1) % 10).

Components - Indicators

  • If the serial number has...
    • 4 digits, set to (indicators).
    • 3 digits, set to (2 * lit indicators).
    • 2 digits, set to (2 * unlit indicators).

Components - Ports

Identify the two ports - Port #1 and Port #2 - on the right side of the module’s device. Set to Port #1, then cycle it to the left (Port #2’s number in the table below) times.

Components - Manuals

  • If destination area is Mercury, Venus, or The Sun, set to 9.
  • Otherwise, starting with 1...
    • If the destination sector is...
      • USA #1, USA #2 or Eurasia, add (2 * port plates).
      • Africa or Rest of America, add (battery holders).
    • If the last digit of the serial number is even, add 1.
  • Set to the resulting number, or 9 if the number exceeds 9.
